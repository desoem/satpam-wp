== Description ==
This plugin provides two-factor authentication for WordPress using the Google Authenticator app.

== Installation ==
1. Upload the `satpam-wp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings via the plugin settings page.
